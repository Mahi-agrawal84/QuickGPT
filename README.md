# 🚀 QuickGPT

QuickGPT is a modern AI-powered chatbot web application that enables users to interact with an intelligent assistant in real time. It provides a clean, responsive interface and delivers fast, conversational responses using an AI language model.

---

## 📌 Features

- 🤖 AI-powered chatbot
- 💬 Real-time conversation interface
- ⚡ Fast response generation
- 🎨 Clean and responsive UI
- 🌙 Modern user-friendly design
- 📱 Mobile-friendly layout
- 🔒 Secure API key management using environment variables

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Mahi-agrawal84/QuickGPT.git
```

### Navigate to Project

```bash
cd QuickGPT
```

### Install Dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

---

## ▶️ Run the Project

Backend

```bash
npm start
```

Frontend

```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder.

```
API_KEY=Your_API_Key
PORT=5000
```
